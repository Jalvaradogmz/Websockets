#Pasos para instalar

* npm install express --save
* npm i socket.io -S
* npm i nodemon --save-dev

cambiar en package.json para que se abra el server con solo escribir un comando

    "scripts": {
        "start": "nodemon server/main.js"
    }

para correr el server se ejecuta con 

    npm start
